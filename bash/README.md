# Basic commands
These programs are available on every  *nix system

### files & directories

**`ls`**  list the files inside the directory   
**`cd`** change directory
**`rm`** deletes file or directory
**`cp`** copies files or directory 
**`mv`** rename or move a file
**`touch`** creates a file
